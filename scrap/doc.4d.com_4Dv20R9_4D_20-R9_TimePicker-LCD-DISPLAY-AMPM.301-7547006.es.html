    TimePicker LCD DISPLAY AMPM 


     

    TimePicker LCD DISPLAY AMPM ( nomObjeto ; amPm ) 
    Parámetro Tipo   Descripción
    nomObjeto  Texto Nombre del objeto subformulario
    amPm  Booleano True = mostrar AM/PM, False = no mostrar

    Descripción   

    El comando TimePicker LCD DISPLAY AMPM muestra u oculta el AM/PM ubicado a la derecha del objeto de subformulario nomObjeto (reloj digita únicamentel) .

    Estas letras se usan para distinguir entre la mañana y la tarde, cuando se utiliza el reloj en el modo 12 horas (ver TimePicker LCD SET MODE).

    Por defecto, se muestran estas letras. Puede pasar False amPm para ocultarlas.

    Ejemplo  

    Queremos ocultar el AM/PM:

     TimePicker LCD DISPLAY AMPM("Subform1";False)
